# SmartPostAI — UI 인벤토리 (클로드 디자인 리디자인용)

> 클로드 디자인으로 새로 만들 때 필요한 **화면 · 구조 · 버튼 · 공통 컴포넌트** 목록.
> 모바일 퍼스트 (컨테이너 `max-w-lg`, 일부 `max-w-sm`).

---

## 1. 공통 컴포넌트 (한 번 디자인 → 전 페이지 재사용)

현재 `components/ui.tsx`에 모여 있음. 클로드 디자인 토큰은 여기만 바꾸면 대부분 전파됨.

| 컴포넌트 | 역할 | 상태/변형 |
|----------|------|-----------|
| **PageLayout** | 페이지 컨테이너 (`max-w-lg`, 좌우 패딩, 상하 여백) | — |
| **PageHeader** | 상단 헤더: `← 돌아가기` + 가운데 제목 | 뒤로가기 유무 |
| **Card** | 흰 카드 (라운드, 보더) | 기본 / 패딩 제거(`!p-0`) |
| **Label** | 폼 라벨 (좌측 텍스트 + 우측 슬롯) | 우측 링크 슬롯 옵션 |
| **Input** | 텍스트 입력 | 기본 / 포커스 / 비활성 |
| **Textarea** | 멀티라인 입력 (높이 가변) + 글자수 카운터 | — |
| **Spinner** | 버튼 내 로딩 스피너 (SVG animate-spin) | — |

## 2. 버튼 인벤토리 (디자인할 변형 = 11종)

| # | 변형 | 용도 (예) | 상태 |
|---|------|-----------|------|
| 1 | **Primary (솔리드, 풀폭, py-4)** | 생성하기 / 변환하기 / 등록하기 / 로그인 | 기본·hover·active·**busy(스피너+문구)**·disabled(회색) |
| 2 | **Secondary (회색 솔리드, py-3)** | 다시 생성하기 / 미리보기로 / 직접 수정 / 취소 / 게시물 복사 | 기본·hover·disabled |
| 3 | **Outline (흰 배경+보더)** | 네이버용 복사 / 티스토리용 복사 / +플랫폼 생성 | 기본·hover·disabled |
| 4 | **Dark (bg-gray-900)** | 슬라이드 영상 만들기 / 영상 저장 | 기본·hover·busy(%표시) |
| 5 | **Danger (빨강)** | 정말 삭제 / 전체 삭제 | 솔리드 / 텍스트형 |
| 6 | **Chip / Pill (선택형)** | 글 스타일·숏폼 스타일 선택 | **선택 / 미선택** |
| 7 | **Segmented Toggle (분절 토글)** | 플랫폼·말투·미리보기 탭 전환 | 2~3분할, 활성/비활성 |
| 8 | **Dashed Add (점선)** | + 제품 추가 / + 샘플 추가 / + 새 스타일 | 기본·hover |
| 9 | **Ghost / Link (텍스트)** | 홈 하단 네비·내 스타일 관리·취소 | 기본·hover |
| 10 | **Icon (× 제거)** | 사진/제품/히스토리/샘플 삭제 | — |
| 11 | **Back (←)** | 헤더 뒤로가기 | — |

## 3. 피드백 / 상태 요소

| 요소 | 스타일 | 용도 |
|------|--------|------|
| **에러 배너** | 빨강 (`bg-red-50` / `text-red-600`) | 생성 실패 등 |
| **경고 배너** | 앰버 (`bg-amber-50` / `text-amber-700`) | 쿠팡 대가성 고지 안내 |
| **성공 토스트** | 초록 텍스트 ("복사됐어요!") | 복사 완료 |
| **뱃지 / 카운터** | pill 형 | 게시물 번호, 플랫폼 라벨, `n/500자`, 글자수 |
| **빈 상태(empty)** | 가운데 회색 안내 + CTA | 히스토리/스타일 없음 |
| **로딩 화면** | "로딩 중..." | result 진입 시 |

---

## 4. 폼 컨트롤

- **텍스트 인풋** (제목, 스타일 이름, 제품명)
- **비밀번호 인풋** (가운데 정렬, `tracking-widest`, `••••`)
- **텍스트영역** + 우측 하단 글자수 (예: `0/500`, `1,234자`)
- **제품+링크 행(row)**: `#n` + 제품명 인풋 + ×, 그 아래 링크 인풋 (파란 텍스트), 반복 + "제품 추가"
- **사진 업로더**: 3열 그리드, 썸네일(+ × 제거 버튼), 마지막에 점선 "추가" 타일 (최대 5장)

## 5. 화면별 구조 (12개 화면)

### 5-1. 로그인 `/login` (`max-w-sm`)
- 타이틀 "SmartPostAI" + 부제
- 비밀번호 인풋 (숫자 키패드) → 에러 배너 → **[들어가기]** (Primary)

### 5-2. 홈 `/` (메인)
- 헤더: 타이틀 + 부제 ("사진만 올려도…🌾")
- **사진 업로더** (최대 5장)
- **메모 인풋** (textarea + 카운터)
- **스타일 선택** (Chip 그룹 + "내 스타일 관리 →" 링크) + 선택 설명
- **발행 플랫폼** (Segmented: 네이버 / 티스토리) + 안내문
- 에러 배너
- 하단 네비(Ghost 링크): **숏폼 만들기 → · 쿠팡 파트너스 → · 기존 글 변환 → · 이전 글 보기 →**
- **[블로그 글 생성하기]** (Primary, busy 상태)

### 5-3. 블로그 결과 `/result`
- 헤더 "미리보기"
- (네이버+티스토리 둘 다일 때) **미리보기 탭** (Segmented)
- **BlogPreview 카드**: 제목 + 본문(HTML 렌더) / 편집모드(제목 인풋 + HTML textarea + "미리보기로 돌아가기") + **[직접 수정하기]**
- (한쪽만 있을 때) **[+ 네이버/티스토리 버전도 생성하기]** (Outline)
- 에러 배너
- **[다시 생성하기]** (Secondary, busy)
- **CopyAndOpen 버튼 묶음** ↓

### 5-4. CopyAndOpen (결과 하단 버튼군)
- **[서식 복사 → 네이버 블로그 열기]** (Primary 초록)
- **[텍스트 복사 → 티스토리 열기]** (Primary 주황)
- 한 줄에 **[네이버용 복사] [티스토리용 복사]** (Outline 2분할)
- **[해시태그 복사 (#n개)]** (Outline)
- 각 버튼 복사 완료 문구로 토글

### 5-5. 기존 글 변환 `/convert`
- 헤더 "기존 글 변환" + 설명
- 큰 textarea + 글자수 → 에러 배너 → **[네이버 + 티스토리 변환하기]** (Primary, busy)

### 5-6. 히스토리 `/history`
- 헤더 "이전 글 목록"
- 우상단 **[전체 삭제]** (텍스트) → 확인 시 **[정말 삭제][취소]**
- 빈 상태: 안내 + "글 생성하러 가기"
- **히스토리 카드 리스트**: 제목 + × 삭제 + 미리보기 2줄 + 스타일·날짜 메타 (탭 시 결과로 이동)

### 5-7. 내 스타일 관리 `/styles`
- 헤더 "내 스타일 관리" + 설명
- **스타일 카드**: 이름 + "스타일 삭제", 샘플 목록(각 삭제), **[+ 샘플 추가 (n/3)]**(점선) / 추가폼(textarea + 추가/취소)
- 빈 상태 안내
- 하단 **[+ 새 스타일 등록하기]**(점선) / 등록폼(이름 인풋 + 샘플 textarea + 등록/취소)

### 5-8. 숏폼 입력 `/shortform`
- 헤더 "숏폼 구성안 만들기" + 설명 🎬
- 사진 업로더 + 메모 인풋
- **플랫폼** (Segmented 3분할: ▶️유튜브 숏츠 / 📸인스타 릴스 / 🎵틱톡)
- **스타일** (Chip: 📌정보형 / 🌙감성형 / 😆유머형) + 설명
- 에러 배너 → **[숏폼 구성안 생성하기]** (Primary, busy)

### 5-9. 숏폼 결과 `/shortform/result`
- 헤더 "{플랫폼} 구성안"
- **제목 카드** (제목 + 플랫폼 뱃지 + ⏱길이 · 🎵BGM)
- **3초 훅 카드** (강조 컬러)
- **컷별 구성 카드 리스트** (번호 뱃지 + 자막 + 🎙멘트 + 🎬화면)
- **유튜브 캡션 카드** (설명 + 해시태그)
- **슬라이드 영상 카드** (ShortformVideo ↓)
- 복사 토스트
- **[전체 대본 복사]** (Primary) · **[캡션 + 해시태그 복사]** (Secondary)

### 5-10. ShortformVideo (영상 생성 위젯)
- **[🎞 슬라이드 영상 만들기]** (Dark, busy=`%`) + **[취소]**(Ghost)
- 완료 시: **비디오 플레이어**(9:16) + **[영상 저장하기]**(Dark) + **[다시 만들기]**(Ghost)
- 미지원 안내 문구 / 에러 텍스트

### 5-11. 쿠팡 파트너스 입력 `/coupang`
- 헤더 "쿠팡 파트너스 글 (쓰레드)" + 설명 🧵
- 사진 업로더 + 메모 인풋
- **제품 + 쿠팡 링크** (반복 행: #n 제품명 + × / 링크 인풋) + **[+ 제품 추가]**(점선) + 안내
- **말투** (Segmented: 반말(바이럴형) / 존댓말)
- **앰버 경고 배너** (대가성 고지 안내)
- 에러 배너 → **[쿠팡 파트너스 글 생성하기]** (Primary 노랑, busy)

### 5-12. 쿠팡 파트너스 결과 `/coupang/result`
- 헤더 "쿠팡 파트너스 스레드" + 안내 🧵 + 복사 토스트
- **게시물 카드 리스트**: `n번째 게시물` 뱃지 + `n/500자`(초과 시 빨강) + 본문(whitespace-pre) + **[이 게시물 복사]**(Secondary)
- **해시태그 카드** + **[해시태그 복사]**
- **[전체 복사 (구분선 포함)]** (Primary)
- 하단 안내 문구

## 6. 현재 색 → 클로드 디자인 결정 필요

| 위치 | 현재 색 | 클로드 전환 시 |
|------|---------|----------------|
| 전역 배경 | `bg-gray-50` | 크림 (#FAF9F5) |
| 네이버 | 초록 (green-500) | ? |
| 티스토리 | 주황 (orange-500) | ? |
| 숏폼 | 빨강 (red-500) | ? |
| 쿠팡 | 노랑 (yellow-400) | ? |
| 영상 | 검정 (gray-900) | ? |
| 포인트 통일 시 | — | 클로드 코랄 (#D97757) |

> **결정 1**: 포인트 색을 코랄로 **전부 통일** vs 배경·폰트만 클로드 + 페이지별 색 유지
> **결정 2**: 헤딩 **세리프**(클로드 느낌) vs 산세리프
> **결정 3**: 다크모드 포함 여부

## 7. 디자인 토큰 (제안 — 통일안 기준)

```
배경      #FAF9F5  (warm cream)
표면/카드  #FFFFFF  + 보더 #E8E5DC
텍스트     #1F1E1D  / 보조 #6B6862
포인트     #D97757  (Claude coral) / hover #C2613F
포인트 약  #F3E9E3  (코랄 배경 틴트)
경고       앰버 계열 유지
위험       #C0392B 계열
라운드     카드 16px / 버튼 12px / 칩 999px
폰트       헤딩=Serif(예: Fraunces) · 본문=Sans(시스템/Inter)
간격       페이지 패딩 16px, 섹션 간 24px
```
