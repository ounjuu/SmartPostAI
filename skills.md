# Skills & Features

SmartPostAI의 주요 기능 및 스킬 명세서입니다.

---

## 1. 사진 업로드

| 항목 | 내용 |
|------|------|
| 컴포넌트 | `components/PhotoUploader.tsx` |
| 최대 업로드 | 5장 |
| 지원 형식 | image/* (JPG, PNG, HEIC 등) |
| 처리 방식 | FileReader → Base64 인코딩 → state 저장 |
| UI | 그리드 미리보기 + 개별 삭제 버튼 |

## 2. 메모 입력

| 항목 | 내용 |
|------|------|
| 컴포넌트 | `components/MemoInput.tsx` |
| 최대 글자 | 500자 |
| 용도 | AI에게 전달할 키워드/메모/상황 설명 |

## 3. AI 블로그 글 생성

| 항목 | 내용 |
|------|------|
| API 라우트 | `app/api/generate/route.ts` |
| AI 클라이언트 | `lib/gemini.ts` |
| 모델 | Google Gemini 1.5 Flash (무료 티어) |
| 입력 | 사진(Base64) + 메모 텍스트 |
| 출력 | JSON `{ title, content }` (HTML 서식 포함) |
| 글 톤 | 친근한 일상 톤 고정 |
| 해시태그 | 글 마지막에 5~10개 자동 생성 |

### 프롬프트 전략

- 네이버 블로그 전문가 역할 부여
- 사진별 자연스러운 설명 포함 지시
- HTML 서식 (`<br>`, `<b>`, `<hr>`) 사용 지시
- 이모지 자연스럽게 포함
- JSON 형식 응답 강제

## 4. 글 미리보기 & 편집

## 5. 서식 복사 + 네이버 블로그 열기

| 항목 | 내용 |
|------|------|
| 컴포넌트 | `components/CopyAndOpen.tsx` |
| 유틸리티 | `lib/clipboard.ts` |
| 복사 방식 | Clipboard API + ClipboardItem (HTML 서식 유지) |
| 앱 열기 | `naverblog://` 딥링크 (모바일) |
| 폴백 | `https://blog.naver.com` (데스크톱/앱 미설치) |

### 복사 플로우

```
"복사하고 네이버 블로그 열기" 클릭
  → HTML 서식 포함 클립보드 복사
  → 0.5초 대기
  → 네이버 블로그 앱 딥링크 실행
  → (앱 미설치 시) 2초 후 웹 폴백
```

---

## 사용자 플로우 요약

```
[메인 페이지]
  사진 업로드 (최대 5장)
  + 메모/키워드 입력
  → "블로그 글 생성하기" 클릭
      ↓
[AI 처리]
  Gemini API가 사진 분석 + 메모 기반 글 생성
      ↓
[결과 페이지]
  생성된 글 미리보기
  → 필요시 직접 수정
  → "복사하고 네이버 블로그 열기" 클릭
      ↓
[네이버 블로그]
  앱/웹에서 붙여넣기(Ctrl+V) → 발행
```

---

## 기술 제약 사항

- **Gemini 무료 티어**: 분당 15회 API 호출 제한
- **이미지 크기**: Base64 인코딩으로 인해 대용량 이미지는 API 전송 시 느릴 수 있음
- **클립보드 API**: 일부 구형 브라우저에서 HTML 복사 미지원 (텍스트 폴백 제공)
- **네이버 블로그 딥링크**: 앱 미설치 시 웹으로 폴백
